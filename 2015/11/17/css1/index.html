CSS盒子模型那些事儿

  • css
  • 前端

盒子模型介绍

  • 块标签都具备盒子模型的特征
  • 宽度width和高度height：盒模型的高度宽度与平时理解不同，一个元素实际宽度（盒子的宽度）=左边距+左边框+左填充+内容宽度+右填充+右边框+右边距。高度类似。 不设置情况下，默认为父容器的100%。
  • 填充padding：元素内容与边框之间的距离。按顺时针设置上、右、下、左 div{ padding:2px 4px 1px 5px; } 或 上下左右填充都相同{ padding:5px; } 或 上下、左右分别相同 { padding:2px 4px; }

    边框

    div{border：2px solid red}等同于
    div{
    border-width:2px;
    border-style:solid; //参数有：dashed(虚线)dotted(点线)solid(实线)
    border-color:red;
    }

盒子大小的计算

一个盒子实际的宽度=左边距+左边框+左填充+内容宽度+右填充+右边框+右边距
如果我们设置一个块级元素的width的值为100px，则这个块级元素的宽度不一定是100px，这个盒子模型的宽度实际宽度还要加上内边距、边框宽度和外边距的两倍。
