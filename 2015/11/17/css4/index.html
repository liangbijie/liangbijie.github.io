css 布局模型那些事儿

  • css
  • 前端

元素的三种布局模型

  • 流动模型(Flow)
  • 浮动模型(Float)
  • 层模型(Layer)

流动模型

流动模型介绍

流动模型是网页中默认的布局模式，即在默认状态下的HTML网页元素都是根据流动模型来分布网页内容的。

流动模型特点

  • 块状元素会在所处的包含元素内自上向下垂直分布（默认状态下，即块状元素的霸道属性）
  • 内联元素会在所处的包含元素内从左到有水平分布

浮动模型

浮动模型介绍

元素浮动会使块状元素不会独霸一行

浮动元素写法

float:left;
float:right;

层模型

层模型介绍

层模型就像 ps图层(幸好学过ps，对此比较了解)一样

层模型类型

  • 绝对定位(position:absolute)
    • 会从文档流脱离出来
    • 可以用left、right、top、bottom属性相对于其最接近的一个具有定位属性的父包含块进行绝对定位（如果不存在，则以body为父元素）
  • 相对定位(position:relative)
    • 没有脱离文档流
    • 通过left、right、top、bottom属性确定元素在正常文档流中的偏移位置。
    • 相对于以前的位置移动，移动的方向和幅度由left、right、top、bottom属性确定，偏移前的位置保留不动。
  • 固定定位(position:fixed) 被固定到了屏幕上了
    • 与absolute定位类型类似，但它的相对移动的坐标是视图（屏幕内的网页窗口）本身
    • 不会随浏览器窗口的滚动条滚动而变化
    • 固定定位的元素会始终位于浏览器窗口内视图的某个位置
    • 不会受文档流动影响
