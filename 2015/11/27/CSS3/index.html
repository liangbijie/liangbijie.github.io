CSS元素类型那些事儿

  • css
  • 前端

html中的元素被分为三种不同的类型：

  • 块状元素
  • 内联元素（行内元素）
  • 内联块状元素 常用的块状元素有：

<\div>、<\p>、<\h1>…<\h6>、<\ol>、<\ul>、<\dl>、<\table>、<\address>、<\blockquote> 、<\form>

常用的内联元素有：

<\a>、<\span>、<\br>、<\i>、<\em>、<\strong>、<\label>、<\q>、<\var>、<\cite>、<\code>

常用的内联块状元素有：

<\img>、<\input>

首先来介绍块级元素

我们可以将其他元素设置成块级元素： display:block;
块级元素一定要记住他的特性：

  • 每个块级元素都会重新新起一行开始，并且其后的元素也会另起一行（霸道属性，独占一行）
  • 元素的高度、宽度、行高以及顶和底边距都可以设置（可塑造属性）
  • 元素宽度在不设置的情况下，宽度和他父元素的宽度一样（依赖属性）

接下来来介绍内联元素

我们可以将其他元素设置成内联元素：display：inline；
内联元素也一样要记住他的特性：

  • 和其他元素都在一行上
  • 元素的高度、宽度及顶部和底部边距不可设置；
  • 元素的宽度就是它包含的文字或图片的宽度，不可改变。
  • 两个内联元素之间会有间距

最后介绍内联块状元素

我们可以将其他元素设置成内联块状元素：display：inline-block;

  • 和其他元素都在一行上；
  • 元素的高度、宽度、行高以及顶和底边距都可设置。
