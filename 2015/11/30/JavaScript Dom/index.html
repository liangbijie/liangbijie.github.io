第8章 充实文档的内容

不应该做什么

此章明确表明我们不应该使用JS把重要的内容添加到网页内容里面去
原因：
第一点：如果用户禁用了JS或者缺乏CSS和DOM支持的用户就看不到其JS添加的重要内容
第二点：各大搜索器机器人也几乎不支持JavaScript，所以也就搜不到你用js添加的内容
本节还讲了我们要遵循渐进增强和平稳退化原则
渐进增强原则：就是从网页最核心的部分（即内容）开始做起，然后再根据内容使用标记良好的结构再逐步加强这些内容
平稳退化原则：其实我认为这个原则和渐进增强说的是一个意思，换汤不换药。如果我们按照渐进增强的原则去充实内容，我们为内容添加的样式和行为就自然支持平稳退化。

把“不可见”变成可以见

html标签中有很多标签有隐藏属性，如title、alt等
有些浏览器会把title属性或者alt属性内容作为弹出式的提示框<这些都是标签的滥用！！！>
alt原用途是：在图片不可见的时候用一段文字来描述这个位置的图片
为了解决某些滥用的行为我们可以用DOM编程来这些问题
方法：

  • 得到隐藏在属性里的信息
  • 创建标记封装这些信息
  • 把这些标记插入到文档

接下来就来说说怎么实现上面步骤

内容

在说步骤之前首先我们来区分一下和的区别
<abbr>含义是“缩略语”abbreviation
<acronym>含义是“首字母缩写词”acronym
所有的首字母缩略词都是缩略语，但所有缩略语不一定都是首字母缩略词
举个例子 list缩写成li这个例子中的li的是缩略语但他不是首字母缩略词
其实很容易记就是只要是缩略了的都可以用<abbr>
告诉大家一个好消息就是：在HTML5中<acronym>已经被<abbr>取代了！！！

选用HTML、XHTML还是HTML5

我觉得对于标记而言，选用HTML还是XHTML是我们的习惯和自由，但是最重要的是不是我们选用哪种文档类型，而是我们使用的标记必须与我们选用的DOCTYPE声明保持一致
下面来介绍下XHTML和HTML5文档的一些要求
XHTML对标记有更严格的要求：

  • 要求所有的标签名和属性名都必须使用小写字母
  • 所有的标签都必须闭合，<img>和<br>等单标签也不例外。
    即应该写成<img />和<br />，为了与早期的浏览器保持兼容，应该在反斜杠字符前保留一个空格。
  • 文档声明
    `<!DOCTYPE html
PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN"
    "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">`

对于XHTML的文档说明我只能说非常非常繁琐，相比之下我更喜欢HTML5的文档说明，详情见11章~

显示“缩略语列表”

本章的任务是把一篇文章里面的<abbr>标签中的title属性和包含的文本集中显示出来，即出现以下的列表

<dl>
    <dt>W3C<dt>
    <dd>World Wide Web Consortium</dd>
    ......
</dl>

我们可以使用DOM来创建这个定义列表，步骤如下：

  1. 遍历<abbr>
  2. 保存<abbr>属性中的title属性
    3.保存<abbr>包含的文本
  3. 创建一个dl
  4. 遍历<abbr>中的title和文本
  5. 创建一个dt
  6. 将文本插入dt
  7. 创建一个dd
  8. 将title插入dd
  9. 循环步骤5-9，直到都插入完

  10. 把dl元素追加到HTML文档的body元素上

    编写displayAbbreviations函数

    此函数就是用来实现以上步骤的
    步骤1.遍历<abbr>

    var abbreviations=document.getElementByTagName(“abbr”);

为了使此函数使用所有的文档，我们需要判断一下是否含有<abbr>

if(abbreviations.length<1)
    return false;

步骤2-3 保存title和文本内容

var defs= new Array(); //保存数据的数组
for(var i=0;i<abbreviations.length;i++){
    var definition=abbreviations[i].getAttribute('title');
    var key=abbreviations[i].lastChild.nodeValue;
    defs[key]=definition;
}

以上三条语句也可以变成一条语句

defs[abbreviations[i].lastChild.nodeValue]=abbreviations[i].getAttribute('title');
