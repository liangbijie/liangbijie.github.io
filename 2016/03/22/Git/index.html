导读：Git的一些常用命令，用来当做手册来查找

Git 常用命令

mkdir 创建一个文件夹
pwd 显示当前目录
git init 把当前目录变成git可以管理的仓库
ls -ah 可以看到隐藏目录 如.git文件
git status 让我们时刻掌握仓库当前的状态,可以随时掌握工作区的状态

$ git status
On branch master
Your branch is up-to-date with 'origin/master'.
Changes to be committed:
  (use "git reset HEAD <file>..." to unstage)
        modified:   src/.idea/workspace.xml
        modified:   src/main/webapp/static/js/page/user/userInfo.js

这段话表示有两个文件被修改了

git diff 可以查看修改的内容(比较当前文件和暂存区文件差异)
git reset --hard HEAD^ 可以回退到上一个版本
git reflog 记录每一次命令操作（可以用来查看命令历史，以便确定要回到未来的哪个版本）
git log 可以看到每一次提交的版本
cat 打开某一个文件
HEAD 表示当前版本
HEAD^ 表示上一个版本
HEAD^^ 表示上上个版本
git reset --hard 3628164 3628164表示版本号，通过git log可以看到版本号，版本号没必要写全，前几位就可以了，Git会自动去找
git checkout -- file 丢弃工作区的修改
git checkout 用版本库里的版本替换工作区的版本
git reset HEAD file 让暂存区的文件到工作区
git remote add origin git@github.com:liangbijie/-test01.git 关联远程库
git push -u origin master 将本地主分支推到远程(如无远程主分支则创建，用于初始化远程仓库)
git remote -v 可以查看本地参考关联的远程仓库地址
git rm 删除文件（+git commit 是删除远程仓库文件； +git checkout – file 误删还原（从远程仓库拉下来） ）
